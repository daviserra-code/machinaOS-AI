MachinaOS Screens

Interface captures of the current runtime: control surfaces, planning and automation views, and execution telemetry.

Featured

The six capabilities defining the May 2026 release: the Visual Workflow Studio, its step-by-step Studio Debugger, chain-to-chain composition, the Agent Communication Graph, the MCP integration surface, and the enhanced Neural Link.

Core Controls

The core control surfaces manage the MachinaOS runtime lifecycle: the home dashboard, the tool registry, the workspace memory browser, the system Settings panel for LLM provider configuration, and the Security Center for the secrets vault, audit log, vulnerability scanner, and credential scrubber.

Automation

The automation surfaces cover Workflow Studio, multi-agent coordination, chain composition, and the MCP integration layer. Build pipelines visually, debug them step by step, compose chains into larger workflows, and connect any Model Context Protocol server as a first-class tool source.

Runtime Insights

The observability surfaces expose execution telemetry in real time. The Event Inspector streams every tool call, approval decision, and agent message. The Metrics dashboard shows plan success rates, tool performance, and SVG charts. The Task Timeline traces every step in a run, and reports consolidate audits and refactors into permanent workspace artifacts.
