System Initialization

Activation Protocol

A guided flow for onboarding teams to a local, intent-driven operating surface.

01

Initialize Core Runtime

Start the local service layer and expose the orchestration API endpoint on your machine.

curl -sL get.machina.os | bash
02

Map Workspace Context

Register your repository path so MachinaOS can resolve tools, files, and process state with deterministic path safety. 

machina workspace connect --path ./project-x
03

Sync Operator Identity

Attach policy preferences for approval gates, execution limits, and logging transparency requirements.

machina profile apply --name local-safe-default

Complete sync

When setup is done, run your first goal with full plan preview and execution trace.

machina run "Prepare my development environment for Project X"
